# Signup Sheet Template

Use a shared sheet for project signup instead of an issue form.

This is the recommended format because students can immediately see:

- which topics already have members
- which topics are full
- which topics still have open slots

## Rules

- Topic assignment is **first-come, first-served**
- **3 students per team is strongly recommended**
- A **4th student is allowed only after prior discussion with the instructor and explicit approval**
- Maximum is **4 students per topic/team**
- Students should type their names directly into an empty slot
- If all four slots are filled, the topic is considered closed unless the instructor changes it

## Recommended columns

- `Topic`
- `Description`
- `Student 1`
- `Student 2`
- `Student 3`
- `Student 4`
- `Status`
- `Notes`

## Suggested status values

- `Open`
- `In progress`
- `Full`
- `Reserved`

## Recommended workflow

1. Instructor creates a Google Sheet, Excel Online sheet, or Notion table.
2. Instructor pastes the topic rows from this template.
3. Students add their names to one of the student columns.
4. Once four students are listed, the instructor marks the row as `Full`.
5. One representative student later submits the final plugin package.
