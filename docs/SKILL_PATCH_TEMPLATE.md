# Skill Patch Template

Students should not submit a complete replacement `SKILL.md`.

Instead, submit a small proposal describing how the orchestrator should use the new tool.

## Minimal template

```md
# Skill Patch Proposal

## Tool
- `my_team_tool`

## Purpose
- Briefly explain what the tool does.

## When to use
- Use this tool when ...

## When not to use
- Do not use this tool when ...

## Modality
- clinical-table / medical-image / fhir / hl7 / note / mixed

## Recommended stage
- post-intake / post-annotation / post-qc / on-demand

## Depends on
- `other_tool_name`
- or `none`

## Approval policy
- approval required / approval not required

## Produces
- `artifact_name`
- `artifact_name`
```
